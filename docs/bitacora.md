# Registro de cambios

Este archivo registra avances principales del Repositorio Accesible.

## Inicio

Se crea el repositorio como espacio digital para organizar materiales disponibles, equipos tecnológicos y materiales elaborados vinculados a la inclusión educativa.

## Documentación

Se agregan documentos de respaldo para sustentar la autoría, finalidad, uso permitido, fuentes, alcance y criterios de registro del proyecto.
